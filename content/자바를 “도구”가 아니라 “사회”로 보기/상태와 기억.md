1. 변수 = 대상 정의 내리기
	1. 기술적  
		프로그램이 값을 저장하거나 참조하기 위해 이름을 붙인 단위이다.  
		예) String name = "김개발";  
  
	2. 사회적  
		인간은 세계의 복잡한 대상과 상태를 이해하고 기억하기 위해 이름을 붙인다.  
		예) 줄기와 가지를 가진 생명체를 "나무"라고 부른다.  
  
	3. 비유의 한계  
		인간의 이름 붙이기는 문화와 맥락에 따라 달라지지만, 변수는 언어 규칙에 따라 엄격하게 동작한다.  
		
2. 타입 = 기억된 것은 어떤 정체성을 가지는가? 
	 1. 기술적 : 값이 어떤 형태로 저장되고  
		어떤 연산이 가능한지를 결정하는 규칙이다.
		EX) INT, BOOLEAN
	 2. 사회적 : 대상을 분류하는 방식이다 
	    EX) 수량, 판단 
	 
3. 연산자 = 작은 함수로 변형을 도와준다 
	1> 기술적 : 값에 어떤 동작을 수행하는 기호
